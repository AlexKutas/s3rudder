# Security Policy

## Supported Versions

Only the latest version of S3Rudder is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.x.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not report it publicly via GitHub issues. 

Instead, please report the vulnerability directly to the project maintainer by emailing **[INSERT EMAIL/CONTACT METHOD]**. 

We will investigate the report and respond as quickly as possible to address the issue.
